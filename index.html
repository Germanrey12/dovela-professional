Dovela Professional

Aplicación de escritorio en Python (Tkinter) — vista estática para Vercel

Descripción

Este repositorio contiene una aplicación de escritorio (interfaz Tkinter) para análisis de dovelas diamante. No es una aplicación web ejecutable en el navegador. Aquí mostramos una landing que explica el proyecto y evita errores 404 comunes (por ejemplo /favicon.ico).

Opciones recomendadas:

  • Descargar el código y ejecutarlo localmente (Python 3.10+ y dependencias del requirements.txt).
  • Crear un front-end web y un backend (FastAPI) si se quiere funcionalidad accesible desde Vercel.

Repositorío en GitHub

Instrucciones rápidas para probar localmente

cd public
python -m http.server 8000

Luego abrir http://localhost:8000

Nota: si quieres que la aplicación funcione como servicio web puedo ayudarte a crear una API (FastAPI) y un front-end React/HTML desplegables en Vercel + Render/ Railway para el backend.
